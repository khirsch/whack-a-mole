# Whack-A-Mole!

#### By [Katrina Hirsch](https://github.com/khirsch)

## Description

Whack-A-Mole! is the final of thirty projects in the [JavaScript30](https://javascript30.com/) series.

## Installation

* `git clone https://github.com/khirsch/whack-a-mole`
* `cd whack-a-mole`
* `open index.html`
